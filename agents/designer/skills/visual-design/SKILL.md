# Visual Design

定义视觉设计系统，包括颜色、字体、组件样式和文案指南。

## 使用场景

- 需要定义产品的视觉风格
- 需要创建轻量设计系统
- 需要编写产品文案指南

## 输入

- PM 文档：BRD（品牌调性）
- UI/UX 规范（如已完成）
- 产品类型和目标用户

## 输出

生成 `docs/design/visual-system.md`，包含：
- 美学方向选择
- 颜色系统
- 字体系统
- 组件样式规范
- 文案指南

## 使用方式

```bash
/visual-design
```
