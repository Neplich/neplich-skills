# Product Requirements Document (PRD)

## Product Overview

A project management SaaS dashboard for small teams (5-20 people).

## Target Users

- Small team leads
- Project managers
- Team members

## Core Features

### 1. Projects View
- List all active projects
- Project status indicators
- Quick access to project details

### 2. Tasks Management
- View tasks by project
- Task status (todo, in progress, done)
- Assign tasks to team members
- Due dates and priorities

### 3. Team Members
- View all team members
- See member availability
- View member workload

### 4. Activity Timeline
- Recent project updates
- Task completions
- Team member activities
- Comments and mentions

## User Stories

- As a team lead, I want to see all projects at a glance
- As a team member, I want to view my assigned tasks
- As a project manager, I want to track team activity
