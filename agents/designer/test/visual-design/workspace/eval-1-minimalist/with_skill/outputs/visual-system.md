# Visual Design System

## 1. Aesthetic Direction
**Minimalist** - Clean, spacious, restrained design for a productivity app targeting professionals.

## 2. Color System

### Primary Colors
- Primary: #2563EB - Professional blue
- Primary Dark: #1E40AF - Hover states
- Primary Light: #DBEAFE - Backgrounds

### Semantic Colors
- Success: #10B981 - Confirmations
- Warning: #F59E0B - Warnings
- Error: #EF4444 - Errors
- Info: #6B7280 - Information

### Neutral Colors
- Text Primary: #111827 - Main text (WCAG AAA)
- Text Secondary: #6B7280 - Secondary text
- Border: #E5E7EB - Dividers
- Background: #FFFFFF - Page background
- Surface: #F9FAFB - Card background


## 3. Typography

### Font Families
- Heading: SF Pro Display - Clean, modern
- Body: SF Pro Text - Readable
- Mono: SF Mono - Code blocks

### Font Scale
- H1: 36px / 44px / 700
- H2: 30px / 38px / 600
- H3: 24px / 32px / 600
- Body: 16px / 24px / 400
- Small: 14px / 20px / 400

## 4. Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px


## 5. Component Styles

### Buttons
- Primary: #2563EB background, white text, 8px padding, 6px radius
- Secondary: white background, #2563EB border, #2563EB text
- Sizes: sm (32px), md (40px), lg (48px)

### Input Fields
- Border: #E5E7EB 1px
- Focus: #2563EB border, subtle shadow
- Height: 40px

## 6. Copy Guidelines

### Voice & Tone
Professional, helpful, clear - no jargon

### Examples
- Buttons: "Create Task", "Save Changes"
- Empty: "No tasks yet. Create your first one!"
- Errors: "Email is required"
